# upstart

  `upstart` wrapper.

## Installation

```bash
  $ npm install configure-upstart
```

## API

#### start(name)

  Start service `name`.

#### stop(name)

  Stop service `name`.

#### restart(name)

  Restart service `name`.

#### reload(name)

  Reload service `name`.

#### enable(name)

  Enable service `name`.

#### disable(name)

  Disable service `name`.
